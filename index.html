Computer Science Quick Start Guide

Setting Up Visual Studio Code for Your First Programming Class

Before You Begin

Welcome! This guide will help you install Visual Studio Code, a free program that lets you write computer programs. You do not need ay programming experience to follow these instrucitons.

  • ⏲️Estimated Time:: 10-15 minutes
  • ⭐Difficulty: Beginner
  • 💻Platform: Windows

What You'll Need

  • Windows computer
  • Internet connection
  • Administrator permissions (if required)

Step 1: Open your internet browser

Click on the program you normally use to browse the internet, such as Google Chrome, Microsoft Edge, or Opera .

Step 2: Go to the Visual Studio Code website

Click on the address bar at the top of the browser window and type code.visualstudio.com. Then press the enter key. You can also just click on the link provided.

Step 3: Download Visual Studio Code

On the homepage, look for the large blue button that says Download for Windows and click the button once.

Your computer will begin downloading the installation file. This may take a few minutes depending on your internet speed.

💡Tip: Do not close your browser while the file is downloading.

Step 4: Open the downloaded file

Double click the downloaded file named something similar to VSCodeUserSetup-x64.exe.
You find the file on the top of your browser on the downloads tab or inside your downloads folder when you open your File Explorer 📁.

Step 5: Allow the installer to make changes

Windows may ask: "Do you want to allow this app to make changes to your device?" in a window similar to the one below.
Click Yes
This is a normal security message.
If you do not see this screen, you can ignore this step.

Step 6: Accept the license agreement

Read the license agreement if you would like.
Click the option that says I accept the agreement.
Then, click Next

Step 7: Continue through the installation

you will reach the page called Select Additional Tasks.
On this page, make sure these are checked:
Add "Open with Code"...
Add to PATH
You can also check Create a desktop icon to find the program on your desktop. This can make it quicker to access the program later but is not required.
Then click Next

💡Tip: Checking these boxes makes Visual Studio Code easier to use later.

Step 8: Install the program

Click Install. Your program will copy the files it needs. This may take a minute or two.

Step 9: Launch Visual Studio

When the Installation finishes, leave the box labeled Launch Visual Studio Code checked.
Then click Finish
Visual Studio Code should open automatically.

Step 10: Verify the installation

If you see a window similar to the one below, congratulations!
Visual Studio Code has been installed successfully.

Troubleshooting

"I can't find the downloaded file."

Open your Downloads folder and look for a file that begins with:"VSCode"


Nothing happens when I double-click.

Right-click the file and select Run as administrator.


The website won't load.

Check that your computer is connected to the internet and refresh the page.

Tips

  • ✅ Read every installation screen before clicking Next.
  • ✅ Restart VS Code if an extension doesn't appear.
  • ✅ Keep your software updated.
